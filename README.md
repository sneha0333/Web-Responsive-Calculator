# Web-Responsive-CalculatorResponsive Web Calculator

A simple and responsive calculator web application built using HTML, CSS, and JavaScript.
This calculator is designed to work smoothly on all screen sizes including mobile, tablet, and desktop devices.

FEATURES 
Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)
Responsive design for all devices
Clean and user-friendly UI
Real-time calculation

CSS
